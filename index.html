Summary: Lattitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accompliosh this analysis, we first pull data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assebling the dataset, we used Matplotlib to plot various aspects of the weather versus latitude. Factors we looked at included temperture, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created by the OpenWeatherMap analysis, as well as explanations and descricptions of any trends and correlations witnessed.
